SAFETG

The Security Auditing Framework and Evaluation Template for Advocacy Groups (SAFETAG) is a professional audit framework and a self-assessment checklist that adapts traditional penetration testing and risk assessment methodoligies to be relevant to small, non-profit, human rights organizations based or operating in the developing world.

Audit Framework

The SAFETAG audit framework has been re-built in a new framework built around open source tools, hosted at github. If you're on Linux, you can follow the installation process to build PDF files out of the colleciton of markdown text documents and prepare audit reports.
